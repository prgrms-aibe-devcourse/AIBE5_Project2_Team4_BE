# AGENTS

## Repository Context

- This repository is treated as a Spring Boot Maven bootstrap project that starts from an effectively empty state.
- The current phase is limited to initial backend setup only.
- Focus on project skeleton, configuration, security/jwt skeleton, and testability.

## Guardrails

- Run `./mvnw test` before finishing work.
- Do not commit secrets or real credentials.
- Do not add Gradle files.
- Do not add server-side template engines such as Thymeleaf, JSP, or Freemarker.
- Keep the scope on bootstrap and environment setup, not full business implementation.
